研究方法论

理论建模能力

  • 高级宏观模型构建：动态优化、递归方法（贝尔曼方程）、跨期决策模型
  • 多目标优化：在约束条件下进行系统性求解与福利分析
  • 不确定性决策建模：风险偏好、时间不一致性、有限理性行为建模

计量实证能力

  • 因果推断：PSM-DID、RDD（断点回归）、IV（工具变量）、DDD（三重差分）、合成控制法
  • 离散选择模型：Logit、Probit及其扩展模型
  • 面板数据建模：固定效应、随机效应、动态面板

计算仿真能力

  • ABM（基于Agent建模）：复杂系统仿真，行为异质性、市场演化、政策效果模拟
  • 强化学习建模：序贯决策优化、多智能体交互仿真
  • 贝叶斯优化：高维参数空间寻优、仿真模型校准
  • 数值仿真：Monte Carlo模拟、Python数值计算

AI工具与研究能力

  • 多Agent协作：熟练运用Claude Code、OpenClaw、Codex、OpenCode、Hermes进行论文辅助写作、文献检索与综述、数据处理与建模分析；搭建个人学术知识库，整合文献管理、笔记整理与检索，提升科研效率
  • OpenClaw"龙虾"：深度开发与定制，打造专属超级远程助手，实现复杂任务自动化处理、多工具调度与自然语言指令理解
  • Claude Code：基于Claude Code 51万行源代码进行深度二次开发，构建个人专属Agent，实现高效代码生成与智能辅助
  • 开发与数据工具：Python、Stata、NetLogo用于数据分析与复杂系统建模；Vscode、Cursor熟练嵌入LaTeX进行学术论文编写与排版；MySQL熟练掌握关系型数据库
